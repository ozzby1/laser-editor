What does this tool do?

Photo Engraving Online converts a regular photo into a rasterized black-and-white image optimized for CO2 or diode laser engraving. You can then import the resulting PNG directly into your laser software (LightBurn, LaserGRBL, XCS, etc.) and engrave it.

Step-by-step guide
1

Upload a photo

Drag the file into the upload area or click to choose it from your device. Supported formats are JPG, PNG, BMP and WebP. For best results, use a sharp photo with good contrast.

2

Set DPI

Choose a resolution that matches your material and your laser’s capabilities. See the recommendation table below.

3

Set the output size in mm

Enter the final engraving width in millimeters. The tool automatically calculates the height, keeps the aspect ratio and shows the resulting pixel count.

4

Fine-tune the width

The slider lets you fine-tune the output width in pixels. It stays synced with the width field in mm, so changing one automatically updates the other.

5

Check the output and download the PNG

The output shows the final rasterized image. If it looks good, click Download PNG and import the file into your laser software.

How to choose the correct output size?

The output size in mm should exactly match the real physical size of the engraving area on your material.

Example: If you want to engrave a portrait on a wooden plate with a width of 80 mm, set the width to 80 mm and choose 300 DPI. The tool will create an image about 945 px wide, which matches the required physical size.

ℹ️ Formula: width in px = (width in mm ÷ 25.4) × DPI

Tips for better results
📷 Input photo quality
Use sharp photos with good contrast. Dark or blurry images will produce worse engraving results even at high DPI.
🔲 Contrast and brightness
Before uploading, try adjusting the photo in any editor. Higher contrast often improves the final rasterized result a lot.
🧪 Test engraving
Always test the result first on an offcut of the same material before running the final engraving.
⚙️ LightBurn settings
Import the PNG as Passthrough or Threshold. The dithering is already included in the file, so additional image processing may make it worse.
